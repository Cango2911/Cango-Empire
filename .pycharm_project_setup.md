## PyCharm Konfiguration

### Python Interpreter
- Verwende das virtuelle Environment: `.venv`
- Python Version: 3.14 (oder höher)

### Git Integration
- Repository ist bereits initialisiert
- Remote: Füge GitHub Repository hinzu

### Run Configurations
Die Scripts können direkt in PyCharm ausgeführt werden:
- `upload_to_docker_www.py` - Upload HTML-Dateien
- `move_sections_correct.py` - Verschiebe News-Sektionen
- `fix_index_html.py` - Korrigiere index.html

## Nächste Schritte

1. **GitHub Remote hinzufügen:**
   ```bash
   git remote add origin https://github.com/USERNAME/Cango-Empire.git
   git push -u origin main
   ```

2. **PyCharm Git Integration:**
   - VCS → Git → Push
   - Oder über GitHub Desktop

3. **Scripts ausführen:**
   - Rechtsklick auf Script → Run
   - Oder Terminal in PyCharm verwenden
